# Quantum Network Presale

## Açıklama
Bu proje, Quantum Network için presale sistemi sunar. NOWPayments ile ödeme alır ve MongoDB ile tokenları kalıcı olarak saklar.

## Kurulum

1. Node.js ve npm yüklü olduğundan emin olun.
2. `.env` dosyasına MongoDB bağlantısını ekleyin.
3. Terminalde proje klasöründe:

```bash
npm install
npm start
